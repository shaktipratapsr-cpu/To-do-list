																																													To-Do List App
A simple Node.js, Express, and MongoDB Atlas to-do list application. Add, view, and update tasks with a clean EJS interface. Data is stored in the cloud for persistence. Great for learning CRUD operations and basic web app structure.

Features:
				Add new tasks with title and description
				View all tasks in a list
				Update existing tasks
				Data stored in MongoDB Atlas
				Minimal, clean UI with EJS

Tech Stack:
				Node.js
				Express.js
				MongoDB Atlas (Mongoose)
				EJS


Getting Started:
				Clone the repository
				Install dependencies
				Configure environment variables:
				Create a .env file in the root directory:
				Start the app:
				Open in your browser:
				Visit http://localhost:5000
